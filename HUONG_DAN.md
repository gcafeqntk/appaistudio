## Bước 1: Chuẩn bị
1.  Tôi đã giúp bạn cài đặt các thư viện cần thiết.
2.  **QUAN TRỌNG:** Bạn cần cấu hình API Key.
    - Mở file `.env.local` trong thư mục này.
    - Sửa dòng `GEMINI_API_KEY=PLACEHOLDER_KEY` thành Key thật của bạn.

## Bước 2: Chạy ứng dụng
Do Node.js của bạn chưa được thêm vào PATH hệ thống, tôi đã tạo một file chạy tắt cho bạn.

1.  Trong thư mục này, tìm file tên là **`run_app.bat`** (màu trắng, có bánh răng).
2.  Click đúp (Double click) vào nó.
3.  Cửa sổ đen sẽ hiện lên và chạy app.
4.  Khi thấy dòng chữ `Local: http://localhost:5173`, hãy giữ phím **Ctrl** và bấm vào link đó để mở web.
